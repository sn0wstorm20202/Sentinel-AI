# Sentinel AI - Keyboard Shortcuts

This document lists all global keyboard shortcuts available in the Sentinel AI Enterprise Fraud Investigation Operating System.

These shortcuts are designed to minimize mouse usage, preserve investigator flow, and maximize triage efficiency.

## Global Navigation
* **`Cmd + K`** (or `Ctrl + K`): Open the Global Command Palette (search cases, entities, devices, and navigate).
* **`Esc`**: Close any open panel (Command Palette, Shortcut Reference, Notification Center, Copilot).
* **`?`**: Open the Keyboard Shortcuts reference dialog.
* **`/`**: Focus the primary search input on the current page (e.g., case queue search).

## Queue & Triage
* **`j`**: Move selection down to the next case in the queue (or open the next case if inside an investigation).
* **`k`**: Move selection up to the previous case in the queue (or open the previous case if inside an investigation).
* **`Enter`**: Open the currently selected case from the queue.

## Investigation Workspaces
*(These shortcuts instantly switch the active tab in the case details view)*
* **`g`**: Switch to **Graph** Intelligence Workspace.
* **`e`**: Switch to **Evidence** & Hypotheses Workspace.
* **`t`**: Switch to Investigation **Timeline** & Audit History.

## Copilot Assistant
* **`c`**: Open the Copilot panel and instantly focus the chat input.

*(Note: If you are actively typing inside an input field or textarea, single-character shortcuts like `j`, `k`, `g`, `c`, `e`, and `t` are automatically disabled to prevent accidental navigation).*
