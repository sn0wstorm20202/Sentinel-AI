# Project Overview (Phase 1)

Sentinel AI is an Enterprise Fraud Intelligence Platform designed to process highly anonymized transactional datasets, identify fraudulent patterns through hybrid machine learning (Tabular + Graph), and expose human-readable intelligence to fraud investigators.

This project goes beyond academic ML (training a model on a CSV). It encompasses full data engineering, feature stores, API development, MLOps, and frontend integration.
